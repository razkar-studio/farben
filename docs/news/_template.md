# Your post title here

Write your post in markdown. The frontmatter `title` is what shows in the
news index; the `# heading` here is what shows when someone clicks through.
Keep them in sync (or omit the heading and let VitePress use the title).

## Subheadings work normally

Code blocks, links, lists — anything markdown supports.

```rust

    println!("hi");
}
```

That's it. Save the file with a `YYYY-MM-DD-slug.md` name in this folder,
commit, push. The index picks it up automatically.
